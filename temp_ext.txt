🎯  Objetivo Principal Prever a direção e o valor futuro da ação ITUB4 com alta acurácia, usando dados históricos e gerando um dashboard interativo. 📊  O que o algoritmo faz passo a passo: 1️ ⃣  Processamento e Limpeza dos Dados Carrega um arquivo CSV (itub4_historico.csv) com dados históricos Detecta automaticamente o  encoding  do arquivo Padroniza nomes de colunas (Date, Close, High,  Low , Open, Volume) Corrige inconsistências ( ex : High <  Low , Close fora das bandas) Remove duplicatas e trata valores nulos com interpolação 2️ ⃣  Criação de Features (Engenharia de Atributos) Cria mais de 50 indicadores técnicos, incluindo: Retornos  em vários períodos (1, 2, 3, 5, 10, 20 dias) Médias móveis  (5, 10, 20, 50, 100, 200 períodos) RSI  ( Relative   Strength  Index) - 7, 14 e 21 períodos Bandas de  Bollinger  (posição, largura) ATR  ( Average   True  Range) - volatilidade Indicadores de volume  (médias,  ratios ) Features temporais  (dia da semana, mês, trimestre) Z-Scores  (padronização estatística) 3️ ⃣  Padronização Z-Score Aplica transformação estatística: Z = (X - μ) / σ Permite comparar features em escalas diferentes Detecta outliers (|Z| > 3) Versão  rolling  (janela móvel de 252 dias) 4️ ⃣  Seleção de Features Usa  Correlação  +  Mutual  Information  para escolher as 30 melhores features Reduz dimensionalidade e melhora performance do modelo 5️ ⃣  Treinamento de 3 Modelos  XGBoost XGBoost  Rápido:  parâmetros simples (100 estimadores, profundidade 4) XGBoost  Base:  parâmetros otimizados (300 estimadores,  early   stopping ) XGBoost  Otimizado:  busca automática de  hiperparâmetros  ( RandomizedSearchCV ) 6️ ⃣  Validação Cruzada Temporal Usa  TimeSeriesSplit  (respeita ordem cronológica) Avalia com MAE (erro absoluto), R² e  acurácia de direção  (sinal correto) 7️ ⃣  Previsões Futuras Gera previsões para os próximos 10 dias Usa o melhor modelo encontrado Projeta preços baseado nos retornos previstos 8️ ⃣  Dashboard Interativo ( Streamlit ) Exibe visualizações: 📈  Gráfico de preço com médias móveis 📊  Evolução dos Z-Scores (com bandas de ±2σ e ±3σ) 🔮  Previsões futuras em cards e gráfico 📉  RSI com zonas de  sobrecompra / sobrevenda 📋  Tabela de Z-Scores atuais com status (outlier/normal) 9️ ⃣  Persistência dos Resultados Salva: itub4_processado_final.csv - dados processados itub4_previsoes_finais.csv - previsões geradas itub4_ metricas.json  - métricas dos modelos 💡  Diferenciais Técnicos Técnica Benefício XGBoost Algoritmo de ponta para séries temporais Early  Stopping Evita  overfitting Z-Score Rolling Adaptação a mudanças de regime TimeSeriesSplit Validação realista (não embaralha dados) Acurácia de Direção Foco em prever se sobe/desce (mais útil que erro absoluto) 📈  Métricas de Sucesso Acurácia de direção:  percentual de acertos na direção do movimento MAE:  erro médio absoluto dos retornos R²:  capacidade explicativa do modelo 🚀  Como Executar bash python  itub4_analise_completa.py O script abre automaticamente um dashboard no navegador (http://localhost:8501). ⚠️  Limitações Depende da qualidade do CSV de entrada Modelo baseado apenas em dados históricos de preço/volume Não incorpora fundamentos (balanços, notícias, macroeconomia) Previsões são probabilísticas, não garantias Resumo final:  É um sistema profissional de  quantitative  trading  que combina engenharia de features financeiras, padronização estatística e  XGBoost  otimizado, entregando previsões e visualizações interativas.
